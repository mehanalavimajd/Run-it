# Runit
a library to run every js file every time that has changed
It's similar to nodemon but runit is faster and lighter
## installation :
```
npm i -g runit
```
## usage
``` 
runit hello.js
```
**note**: If the file extention of the file wasn't .js you would see an error
list of arguments you can give to runit:
```
    --help : shows all arguments that you can give to runit
    --version : shows version
```
