# JsWatch
a library to run every js file every time that has changed
It's similar to nodemon but jswatch is faster and lighter
## installation :
```
npm i -g jswatch
```
## usage
``` 
jswatch hello.js
```
**note**: If the file extention of the file wasn't .js you would see an error
list of arguments you can give to jswatch:
```
    --help : shows all arguments that you can give to jswatch
    --version : shows version
```
